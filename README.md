# 🔍 SISTEMA DE BÚSQUEDA SEMÁNTICA - BoW vs TF-IDF

## 📋 Descripción
Proyecto que implementa y compara dos métodos de representación de texto para búsqueda documental:
- **Bag of Words (BoW)**: Representación basada en frecuencia de términos
- **TF-IDF**: Ponderación que considera importancia relativa de términos

## 🚀 Características
- ✅ Carga automática de documentos .txt
- ✅ Vectorización con BoW y TF-IDF
- ✅ Búsqueda por similitud coseno
- ✅ Comparación visual de resultados
- ✅ Stop words en español personalizadas
